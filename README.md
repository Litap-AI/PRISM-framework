# 🚀 PRISM Framework

## AI-Driven Product Evaluation and Prioritization System

PRISM Framework is an intelligent AI-powered decision support system designed to evaluate, classify, prioritize, and rank products based on strategic business and technological factors.

The framework integrates:
- Rule-based PRISM scoring
- Artificial Neural Networks (ANN)
- Explainable analytics
- Interactive Streamlit dashboard
- Batch CSV product analysis
- Visualization-driven decision intelligence

---

# 📌 PRISM Components

| Component | Description |
|---|---|
| **P** | Performance |
| **R** | Relevance |
| **I** | Innovation |
| **S** | Scalability |
| **M** | Monetization |

These parameters collectively determine product potential and strategic viability.
